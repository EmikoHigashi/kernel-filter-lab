# Interactive Kernel Filter Lab (Web)
https://emikohigashi.github.io/kernel-filter-lab/

画像処理の基本である「カーネルフィルタ（畳み込み演算）」の仕組みを視覚的に学習・体験できるWebツールです。

## 🚀 使い方
1. **画像の読み込み**:
   -  「ファイルを選択」好きな画像をローカルPCからアップロードします。  
👇の画像をダウンロードして使っていただくことができます。  
   <img src="https://github.com/EmikoHigashi/kernel-filter-lab/blob/main/img/%E3%82%B9%E3%82%AF%E3%83%AA%E3%83%BC%E3%83%B3%E3%82%B7%E3%83%A7%E3%83%83%E3%83%88%202026-04-07%20111538.png" alt="Flowers on a Plate" width = 100>　
  
2. **フィルタの適用**:  
   -  kernel Sum の下にある3x3　グリッドがkernel filterです
   -  プルダウンメニューからプリセット（Sharpen, Edge等）を選択します。
   -  3x3　グリッドの数値は自分で変えることができます
   -  kernel Sumは3x3　グリッドの合計値です
     
4. **ピクセルの観察**:
   - 画像の中の赤枠は、**矢印キー**で移動できます。
   - 「計算前」と「計算後」の ピクセル値がリアルタイムに表示されます。
   - 値の範囲は0 ～ 255です。

## 🛠 搭載されているフィルタ
- **Identity**: 何もしない（元の画像）
- **Box Blur**: 画像をぼかします。
- **Sharpen**: 輪郭を強調してくっきりさせます。
- **Edge**: 輪郭のみを抽出します。
- **Emboss**: 浮き出し（エンボス）効果を与えます。

## 💡 技術スタック
- HTML5 / CSS3
- JavaScript (Canvas APIを使用したピクセル操作)
- GitHub Pages (ホスティング)

## 📝 開発の背景
子供向けプログラミング学習で、画像処理における畳み込み演算（Convolution）を教えるために作りました。
