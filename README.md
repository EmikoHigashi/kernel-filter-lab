# Interactive Kernel Filter Lab (Web)
https://emikohigashi.github.io/kernel-filter-lab/

画像処理の基本である「カーネルフィルタ（畳み込み演算）」の仕組みを視覚的に学習・体験できるWebツールです。Python (Tkinter) 版のプロトタイプをベースに、ブラウザで動くように移植しました。

## 🚀 使い方
1. **画像の読み込み**: 「ファイルを選択」から好きな画像をアップロードします。
2. **フィルタの適用**: 右側の 3x3 グリッドの数値を変更するか、プリセット（Sharpen, Edge等）を選択します。
3. **ピクセルの観察**: 
   - 画像上をクリックするか、**矢印キー**で注目するピクセルを移動できます。
   - 赤枠で囲まれた部分の「計算前」と「計算後」の数値の変化がリアルタイムに表示されます。

## 🛠 搭載されているフィルタ
- **Identity**: 何もしない（元の画像）
- **Box Blur**: 画像をぼかします。
- **Sharpen**: 輪郭を強調してくっきりさせます。
- **Edge**: 輪郭のみを抽出します。
- **Emboss**: 浮き出し（エンボス）効果を与えます。

## 💡 技術スタック
- HTML5 / CSS3
- JavaScript (Canvas APIを使用したピクセル操作)
- GitHub Pages (ホスティング)

## 📝 開発の背景
デジタル画像処理における畳み込み演算（Convolution）を直感的に理解するために作成されました。
